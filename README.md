## Instalação


### BACKEND

```sh
cd backend/
```

#### Install the required libraries:
```sh
npm install
```

#### Run the script:
```sh
npm run start
```

## FRONTEND
```sh
cd frontend/
```
#### Install the required libraries:
```sh
npm install
```

#### Run the script:
```sh
npm run start
```
